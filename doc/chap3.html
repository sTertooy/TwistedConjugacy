3 Coincidence and Fixed Point Groups

3.1 Coincidence and Fixed Point Groups

3.1-1 CoincidenceGroup
‣ CoincidenceGroup( hom1, hom2 )( operation )

Let hom1 and hom2 be group homomorphisms from a group H to a group G. This command returns the subgroup of H consisting of the elements h for which h^hom1 = h^hom2.

3.1-2 FixedPointGroup
‣ FixedPointGroup( endo )( operation )

Let endo be an endomorphism of a group G. This command returns the subgroup of G consisting of the elements fixed under the endomorphism endo.

gap> G := AlternatingGroup( 6 );;
gap> H := SymmetricGroup( 5 );;
gap> phi := GroupHomomorphismByImages( H, G, [ (1,2)(3,5,4), (2,3)(4,5) ],
>  [ (1,4)(3,6), () ] );; 
gap> psi := GroupHomomorphismByImages( H, G, [ (1,2)(3,5,4), (2,3)(4,5) ],
>  [ (1,2)(3,4), () ] );;
gap> CoincidenceGroup( phi, psi );
<permutation group with 60 generators>
gap> khi := GroupHomomorphismByImages( G, G, [ (1,2,3,4,5), (4,5,6) ],
>  [ (1,2,6,3,5), (1,4,5) ] );;
gap> FixedPointGroup( khi );
Group([ (), (1,6,5,2,4), (1,5,4,6,2), (1,2,6,4,5), (1,4,2,5,6) ])
