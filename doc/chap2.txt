  [33X[0;0YPlease  note  that  currently  the functions in this chapter are implemented
  only for (homomorphisms of) finite groups and (endomorphisms of) pcp-groups.[133X
  
  
  [1X2.1 [33X[0;0YDouble Twisted Conjugation Action[133X[101X
  
  [33X[0;0YLet  [23XG,H[123X  be  groups and [23X\varphi,\psi: H \to G[123X group homomorphisms. Then the
  pair  [23X(\varphi,\psi)[123X induces a (right) group action on [23XG[123X given by [23XG \times H
  \to G: (g,h) \mapsto g \cdot h = \psi(h)^{-1} g\varphi(h)[123X. This group action
  is  called  [13X[23X(\varphi,\psi)[123X-twisted  conjugation[113X,  and induces an equivalence
  relation on the group. We say that [23Xg_1, g_2 \in G[123X are [23X(\varphi,\psi)[123X-twisted
  conjugate,  denoted  by  [23Xg_1  \sim_{\varphi,\psi}  g_2[123X, if and only if there
  exists some element [23Xh \in H[123X such that [23Xg_1 \cdot h = g_2[123X, or equivalently [23Xg_1
  = \psi(h) g_2 \varphi(h)^{-1}[123X.[133X
  
  [1X2.1-1 TwistedConjugation[101X
  
  [33X[1;0Y[29X[2XTwistedConjugation[102X( [3Xhom1[103X, [3Xhom2[103X ) [32X operation[133X
  
  [33X[0;0YImplements  the twisted conjugation (right) group action induced by the pair
  of homomorphisms ( [3Xhom1[103X, [3Xhom2[103X ).[133X
  
  [1X2.1-2 IsTwistedConjugate[101X
  
  [33X[1;0Y[29X[2XIsTwistedConjugate[102X( [3Xhom1[103X, [3Xhom2[103X, [3Xg1[103X, [3Xg2[103X ) [32X operation[133X
  
  [33X[0;0YTests  whether the elements [3Xg1[103X and [3Xg2[103X are double twisted conjugate under the
  twisted conjugacy action of the pair of homomorphisms ( [3Xhom1[103X, [3Xhom2[103X ).[133X
  
  [1X2.1-3 RepresentativeTwistedConjugation[101X
  
  [33X[1;0Y[29X[2XRepresentativeTwistedConjugation[102X( [3Xhom1[103X, [3Xhom2[103X, [3Xg1[103X, [3Xg2[103X ) [32X operation[133X
  
  [33X[0;0YComputes an element that maps [3Xg1[103X to [3Xg2[103X under the twisted conjugacy action of

  [1X2.2 [33X[0;0YReidemeister Coincidence Classes[133X[101X
  
  [33X[0;0YThe  equivalence classes of the equivalence relation [23X\sim_{\varphi,\psi}[123X are
  called  the  [13XReidemeister  coincidence  classes  of  [23X(\varphi,\psi)[123X[113X  or  the
  [13X[23X(\varphi,\psi)[123X-twisted  conjugacy  classes[113X. We denote the Reidemeister class
  of  [23Xg  \in  G[123X  by [23X[g]_{\varphi,\psi}[123X. The number of Reidemeister coincidence
  classes is called the Reidemeister coincidence number [23XR(\varphi,\psi)[123X and is
  always a positive integer or infinity.[133X
  
  [1X2.2-1 ReidemeisterClass[101X
  
  [33X[1;0Y[29X[2XReidemeisterClass[102X( [3Xhom1[103X, [3Xhom2[103X, [3Xg[103X ) [32X operation[133X
  [33X[1;0Y[29X[2XTwistedConjugacyClass[102X( [3Xhom1[103X, [3Xhom2[103X, [3Xg[103X ) [32X operation[133X
  
  [33X[0;0YCreates  the  Reidemeister  coincidence class of the pair of homomorphisms (
  [3Xhom1[103X,  [3Xhom2[103X  )  [23XH  \to G[123X with representative [3Xg[103X. The following attributes and
  operations are available:[133X
  
  [30X    [33X[0;6Y[10XRepresentative[110X, which returns [3Xg[103X,[133X
  
  [30X    [33X[0;6Y[10XGroupHomomorphismsOfReidemeisterClass[110X, which returns [ [3Xhom1[103X, [3Xhom2[103X ],[133X
  
  [30X    [33X[0;6Y[10XActingDomain[110X, which returns the group H,[133X
  
  [30X    [33X[0;6Y[10XFunctionAction[110X, which returns the twisted conjugacy action on G,[133X
  
  [30X    [33X[0;6Y[10XRandom[110X,  which  returns a random element belonging to the Reidemeister
        class,[133X
  
  [30X    [33X[0;6Y[10X\in[110X,  which  can  be  used  to  test  if  an  element  belongs  to the
        Reidemeister  class  -  only  guaranteed  to  work if the Reidemeister
        number R( [3Xhom1[103X, [3Xhom2[103X ) is finite,[133X
  
  [30X    [33X[0;6Y[10XAsList[110X,  which  lists  all  elements  in the Reidemeister class - only
        works for finite groups.[133X
  
  [30X    [33X[0;6Y[10XSize[110X,  which  gives the number of elements in the Reidemeister class -
        only works for finite groups.[133X
  
  [1X2.2-2 ReidemeisterClasses[101X
  
  [33X[1;0Y[29X[2XReidemeisterClasses[102X( [3Xhom1[103X, [3Xhom2[103X ) [32X operation[133X
  [33X[1;0Y[29X[2XTwistedConjugacyClasses[102X( [3Xhom1[103X, [3Xhom2[103X ) [32X operation[133X
  
  [33X[0;0YReturns  a  list  containing the Reidemeister coincidence classes of ( [3Xhom1[103X,

  [10Xfail[110X otherwise. It is guaranteed that the Reidemeister class of the identity
  is in the first position.[133X
  
  [1X2.2-3 ReidemeisterNumber[101X
