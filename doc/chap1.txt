  [1X1.1 [33X[0;0YTwisted Conjugation Action[133X[101X
  
  [33X[0;0YLet  [23XG[123X be a group and [23X\varphi: G \to G[123X an endomorphism. Then [23X\varphi[123X induces
  a (right) group action on [23XG[123X given by [23XG \times G \to G: (g,h) \mapsto g \cdot
  h  =  h^{-1}  g\varphi(h)[123X.  This  group  action  is  called  [13X[23X\varphi[123X-twisted
  conjugation[113X,  and  induces an equivalence relation on the group. We say that
  [23Xg_1,  g_2 \in G[123X are [23X\varphi[123X-twisted conjugate, denoted by [23Xg_1 \sim_{\varphi}
  g_2[123X,  if and only if there exists some element [23Xh \in G[123X such that [23Xg_1 \cdot h
  = g_2[123X, or equivalently [23Xg_1 = h g_2 \varphi(h)^{-1}[123X.[133X
  
  [1X1.1-1 TwistedConjugation[101X
  
  [33X[1;0Y[29X[2XTwistedConjugation[102X( [3Xendo[103X ) [32X operation[133X
  
  [33X[0;0YImplements  the  twisted  conjugation  (right)  group  action induced by the
  endomorphism [3Xendo[103X. This is the twisted conjugacy analogue of [10XOnPoints[110X.[133X
  
  [1X1.1-2 IsTwistedConjugate[101X

  [33X[0;0YTests whether the elements [3Xg1[103X and [3Xg2[103X are twisted conjugate under the twisted

  [1X1.2 [33X[0;0YReidemeister Classes[133X[101X
  
  [33X[0;0YThe  equivalence  classes  of  the  equivalence  relation [23X\sim_{\varphi}[123X are
  called  the [13XReidemeister classes of [23X\varphi[123X[113X or the [13X[23X\varphi[123X-twisted conjugacy
  classes[113X.  We  denote the Reidemeister class of [23Xg \in G[123X by [23X[g]_{\varphi}[123X. The
  number  of Reidemeister classes is called the Reidemeister number [23XR(\varphi)[123X
  and is always a positive integer or infinity.[133X
  
  [1X1.2-1 ReidemeisterClass[101X
  
  [33X[1;0Y[29X[2XReidemeisterClass[102X( [3Xendo[103X, [3Xg[103X ) [32X operation[133X

  [33X[0;0YCreates  the  Reidemeister  class  of an endomorphism [3Xendo[103X of a group G with
  representative [3Xg[103X. The following attributes and operations are available:[133X
  
  [30X    [33X[0;6Y[10XRepresentative[110X, which returns [3Xg[103X,[133X
  
  [30X    [33X[0;6Y[10XGroupHomomorphismsOfReidemeisterClass[110X, which returns a list containing
        [3Xendo[103X  and  the identity map on G (to be compatible with double twisted
        conjugacy classes),[133X
  
  [30X    [33X[0;6Y[10XActingDomain[110X, which returns the group G,[133X
  
  [30X    [33X[0;6Y[10XFunctionAction[110X,  which returns the twisted conjugacy action of [3Xendo[103X on
        G,[133X
  
  [30X    [33X[0;6Y[10XRandom[110X,  which  returns a random element belonging to the Reidemeister
        class,[133X
  
  [30X    [33X[0;6Y[10X\in[110X,  which  can  be  used  to  test  if  an  element  belongs  to the

  [1X1.3 [33X[0;0YReidemeister Spectra[133X[101X
  
  [33X[0;0YThe  set  of  all  Reidemeister  numbers  of  automorphisms  is  called  the
  [13XReidemeister spectrum[113X and is denoted by [23X\mathrm{Spec}_R(G)[123X, i.e.[133X
  
  
  [24X[33X[0;6Y\mathrm{Spec}_R(G) := \{ R(\varphi) \mid \varphi \in \mathrm{Aut}(G)\}.[133X
  
  [124X
  
  [33X[0;0YThe  set of all Reidemeister numbers of endomorphisms is called the [13Xextended
  Reidemeister spectrum[113X and is denoted by [23X\mathrm{ESpec}_R(G)[123X, i.e.[133X
  
  
  [24X[33X[0;6Y\mathrm{ESpec}_R(G) := \{ R(\varphi) \mid \varphi \in \mathrm{End}(G)\}.[133X
  
  [124X

  [1X1.4 [33X[0;0YZeta Functions[133X[101X
  
  [33X[0;0YLet  [23X\varphi: G \to G[123X be an endomorphism such that [23XR(\varphi^n) < \infty[123X for
  all  [23Xn \in \mathbb{N}[123X. Then the Reidemeister zeta function [23XR_{\varphi}(z)[123X of
  [23X\varphi[123X is defined as[133X
  
  
  [24X[33X[0;6YR_{\varphi}(z) := \exp \sum_{n=1}^\infty R(\varphi^n) \frac{z^n}{n}.[133X
  
  [124X

  of finite groups.[133X
