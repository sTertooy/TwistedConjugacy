5.1-1 TwistedConjugation
‣ TwistedConjugation( endo )( attribute )

Implements the twisted conjugation (right) group action induced by the endomorphism endo. This extends the function OnPoints to twisted conjugation.

5.1-2 ReidemeisterClasses
‣ ReidemeisterClasses( endo )( attribute )
‣ TwistedConjugacyClasses( endo )( attribute )

Returns a list containing the Reidemeister classes of endo if the Reidemeister number of endo is finite, and returns fail otherwise. It is guaranteed that the Reidemeister class of the identity is in the first position. This extends the function ConjugacyClasses to twisted conjugation.

5.1-3 ReidemeisterNumber
‣ ReidemeisterNumber( endo )( attribute )
‣ NrTwistedConjugacyClasses( endo )( attribute )

Returns the Reidemeister number of endo, i.e. the number of Reidemeister classes. This extends the function NrConjugacyClasses to twisted conjugation.

5.1-7 ReidemeisterZetaCoefficients
‣ ReidemeisterZetaCoefficients( endo )( attribute )

For a finite group, the sequence of Reidemeister numbers of the iterates of endo, i.e. the sequence R(endo), R(endo^2), ..., is periodic (see [Fel00, Theorem 16]). This function returns a list containing the first period of this sequence.

5.1-8 TwistedConjugacyClassZeta
‣ TwistedConjugacyClassZeta( endo )( attribute )
‣ ConjugacyClassZeta( G )( attribute )

Returns the twisted conjugacy class zeta function of endo. Note that ConjugacyClassZeta(G) is the same as TwistedConjugacyClassZeta(IdentityMapping(G)).

5.1-9 TwistedConjugacyClassZetaCoefficients
‣ TwistedConjugacyClassZetaCoefficients( endo )( attribute )
‣ ConjugacyClassZetaCoefficients( G )( attribute )

This function returns a list such that the i-th entry in the list is the number of Reidemeister classes of endo of size i. The list always has length equal to the size of the group G. Note that ConjugacyClassZetaCoefficients(G) is the same as TwistedConjugacyClassZetaCoefficients(IdentityMapping(G)).

5.1-10 FixedPointGroup
‣ FixedPointGroup( endo )( attribute )

Let endo be an endomorphism of a group G. This command returns the subgroup of G consisting of the elements fixed under the endomorphism endo. This command is implemented only for endomorphisms of finite groups and of abelian groups.

5.2-1 IsTwistedConjugate
‣ IsTwistedConjugate( endo, g1, g2 )( operation )

Tests whether the elements g1 and g2 are twisted conjugate under the twisted conjugacy action of the endomorphism endo. This extends the function IsConjugate to twisted conjugation.

5.2-2 RepresentativeTwistedConjugation
‣ RepresentativeTwistedConjugation( endo, g1, g2 )( operation )

Computes an element that maps g1 to g2 under the twisted conjugacy action of the endomorphism endo and returns fail if no such element exists. This extends the function RepresentativeAction to twisted conjugation.

5.3 TwistedConjugacy automatic generated documentation of global functions

5.3-1 InducedEndomorphism
‣ InducedEndomorphism( epi, endo )( function )

Let endo be an endomorphism of a group G and epi be an epimorphism from G to a group H such that the kernel of epi is fixed under endo. This command returns the endomorphism of H induced by endo via epi, that is, the endomorphism of H which maps g^epi to (g^endo)^epi, for any element g of G.
